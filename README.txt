Саша и Женя
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Игра-платформер для развития мелкой моторики и скорость реакции с помощью сюжета и цели дойти до "приза".

Управление: осуществляется с помощью стрелок.

Прдеистория: вы рыцарь и обычный овало-человек по имени Женя.
Ваша цель - спасти важного члена правительства и вашего друга Сашу из лап мошенников и заговорщиков.
Найдите клетку с Сашей и разбейте её!

Для запуска игры запустите файл "game.py".

Геймплей находится в видеофайле "GamePlay.mp4".
